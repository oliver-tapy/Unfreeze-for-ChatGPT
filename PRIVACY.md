# Privacy Policy — Unfreeze for ChatGPT

**Last updated:** February 28, 2026

## What this extension does

Unfreeze for ChatGPT intercepts ChatGPT conversation API responses in your browser to reduce the number of messages rendered on screen. This makes long conversations (1000+ messages) load instantly instead of freezing your browser.

## Data collection

**This extension does not collect, transmit, or store any personal data.**

- All processing happens locally in your browser
- No data is sent to any external server
- No analytics, tracking, or telemetry
- No cookies are set by the extension

## Data stored locally

The extension stores one setting in Chrome's local storage:

- `maxMessages` — the number of recent messages to display (default: 250)

This value never leaves your browser.

## Permissions used

- **storage** — to save user preferences locally
- **Host access to chatgpt.com** — to inject the content script that intercepts API responses
